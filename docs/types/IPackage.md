# Package
<a href="../../src/types/PackageManagers.ts">Source file</a>
`Package` is an interface which is used to structure information about each individual package from a package manager.

## Usage
| Name       | Type      |
| ---------- | --------- |
|  `name`    | `string`  |
| `summary?` | `string`  |
| `version`  | `string`  |
| `group?`   | `string`  |
